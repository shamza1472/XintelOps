# XIntelOps Posting Cadence (Operator Mode)

Manual posting only — copy from the operator brief email. Never auto-post.

## Read the email in this order (< 3 minutes)

1. **Operator decisions** — Post now / Watch / Everyone is missing
2. **Top signals today** — ranked list with scores and recommended action
3. **Post now** — only if action is X POST or X THREAD
4. **Journalist engagement** — only if not skipped

## Recommended actions → what to do

| Action | Operator move |
|---|---|
| `X POST` | Post single tweet within 30 min |
| `X THREAD` | Post numbered thread within 30 min |
| `LINKEDIN` | Post Mon/Wed/Fri 09:00–11:00 PKT |
| `MONITOR` | Add to watchlist — no post |
| `ARCHIVE` / `IGNORE` | No action |
| `HIGH PRIORITY TRACKING` | Track over 7–30 days |
| `NEWSLETTER CANDIDATE` | Save for long-form later |

## Daily rhythm

| Slot | What | When |
|---|---|---|
| Primary X | `one_signal_to_post` draft | Within 30 min of email |
| Secondary X | "Everyone is missing" or rank #2 angle | 4–6 hours later |
| Journalist reply | Specific post URL in email | Once daily max |
| LinkedIn | Signal with LINKEDIN action | Mon/Wed/Fri 09:00–11:00 PKT |

## Crisis mode

When any ranked signal has `crisis_flag=true`:

1. Post the primary X action within 15 minutes
2. Defer secondary and journalist engagement until primary is live

## Niche priority (scoring weight)

**Tier 1:** China, Pakistan, India, Gulf, Iran, Levant, Red Sea, Horn of Africa

**Tier 2:** Central Asia, Caucasus, East Africa, ASEAN

**Tier 3:** Everything else — only when it materially impacts Tier 1 or 2
